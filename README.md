# Generador de Vouchers WiFi

Aplicación en Python con interfaz gráfica (Tkinter) que:
- Lee códigos desde un PDF
- Inserta los códigos en un diseño de voucher
- Genera un PDF listo para imprimir con múltiples vouchers por página

## Requisitos
- Python 3.9+
- Poppler (necesario para pdf2image)

## Instalación
```bash
pip install -r requirements.txt
